Использование:
xrCompress.exe <папка_для_сжатия> -ltx <файл_настроек.ltx> <параметры>

Параметры:
-fast   	- быстрое сжатие для форматов: xml, ltx, script, ogf, omf, h, s, ps, vs.
-store  	- просто пакует в db-фархив. Без сжатия.
-strong 	- режим сильного сжатия, кроме форматов выше, сжимает: dds, cform, details,
 env_mod, fog_vol, game, hom, ps_static, spawn,
	но требует больше времени.

В архиве уже настроенные батники:
"! compress_map.cmd"	 - для упаковки мультиплеерных карт.
"! compress_gamedata.cmd" 	 - для упаковки любых игровых ресурсов\модов

Максимальный размер db-архива 850мб, если необходимо упаковать больший объем файлов, то
будут созданы "gamedata.pack_0", "gamedata.pack_1" и т.д. 
".pack_" потом заменить на ".db"

Если для упаковки использовался файл настроек "build_map.ltx",
то архив будет называтся <имя_уровня.db> в соответствии с параметром level_name

Формат ltx:
[options] 
exclude_exts = *.ncb,*.scc,*.vssscc,*.bmp,*.uvm,*.log,*.psd,*.swatch 	; расширения файлов, которые не будут упакованы

[include_folders] 			; папки, которые будут упакованы 
.\ 	= true 			; корень (все файлы и подпапки)

[exclude_folders]			; папки, которые не надо паковать
;scripts	= true			;

[header]
auto_load	     = true			; читать ли этот архив при загрузке игры
level_name   = single		; название уровня, для мода оставить single
level_ver       = 1.0			; версия уровня
entry_point  = $fs_root$\gamedata\ 	; точка входа, не изменять!
creator          = ""			; автор карты 
link                = "www.zergo.ucoz.ua"	; ссылка на скачивание карты, если её нет у других игроков
